# 基于K-means的文本聚类

## 数据集：THUCNews 中文新闻数据集（去掉标签，仅评估时用作真值，根据自己电脑情况，不要求用全部数据，可取部分的类别子集作为实验）。
## 评估指标：
- `轮廓系数（Silhouette）`
- `聚类纯度（Purity）`
- `归一化互信息（NMI）`
### 说明
```text
用 TF-IDF 或词向量表示文档并做 K-means 聚类，重点探讨影响结果的两个因素——K 值的选择（肘部法 / 轮廓系数）与初始簇中心的选择
聚类完成后再结合人工总结为各簇赋予标签。
```
## 合理性能参考
- `类别清晰的语料上 Purity ≥ 0.65`
- `NMI ≥ 0.45 为合格`
- `轮廓系数应为正`
- `需用肘部图说明 K 值依据`
